# 示例素材

当用户只是想体验 GhostCut API 流程、但暂时没有可用视频或图片时，可以提示用户使用下面的公开示例素材 URL。不要把它作为业务默认值，也不要在用户已经提供素材时替换用户素材。

## 视频 URL

```text
https://gc100.cdn.izhaoli.cn/demo/1691660164246.mp4
```

用途：

- 体验文件无需上传时的普通单视频接口流程。
- 测试签名、创建任务和 `/work/status` 查询链路。
- 演示 `videoUrl` / `urls` 这类字段如何填写。

限制：

- 这是短公开视频素材，不代表用户真实业务内容。
- 它不适合验证 ASR、配音、角色识别、字幕翻译等依赖语音或字幕质量的能力。
- 如果功能依赖人声、字幕或剧集素材上下文，应提醒用户换成自己的视频，或先上传本地素材。
- 如果该 URL 后续不可访问，应让用户提供自己的视频，或重新选择一个可直接访问的 `.mp4` URL。

## 图片 URL

```text
https://gc100.cdn.izhaoli.cn/ve_material_image/A-00b4943646344cfc/e2d6dd82903f4059bd94c9b660f5f7b6/1782789120948.webp
```

用途：

- 体验图片文字擦除、图片翻译和图片合成接口流程。
- 测试图片任务创建、`/image/translate/query` 查询和结果读取链路。
- 演示 `downloadInfo` 中图片 URL 的填写方式。

限制：

- 这是公开示例图片，不代表用户真实业务内容。
- 如果用户需要验证真实商品图、海报图或多语言排版效果，应换成用户自己的图片。
- 如果该 URL 后续不可访问，应让用户提供自己的图片，或重新选择一个可直接访问的图片 URL。

## Agent 使用规则

- 只有当用户明确要体验功能且没有素材时，才主动提到示例素材 URL。
- 生成请求示例时，把示例素材标注为“演示 URL，可替换”。
- 不要将示例素材写进固定脚本默认值。
- 调用前可用 `curl -I` 或等价方式确认 URL 可访问。
