# 示例素材

当用户只是想体验 GhostCut API 流程、但暂时没有可用视频时，可以提示用户使用下面的公开视频 URL。不要把它作为业务默认值，也不要在用户已经提供素材时替换用户素材。

## 视频 URL

```text
https://gc100.cdn.izhaoli.cn/demo/1691660164246.mp4
```

用途：

- 体验文件无需上传时的普通单视频接口流程。
- 测试签名、创建任务和 `/work/status` 查询链路。
- 演示 `videoUrl` / `urls` 这类字段如何填写。

限制：

- 这是短公开视频素材，不代表用户真实业务内容。
- 它不适合验证 ASR、配音、角色识别、字幕翻译等依赖语音或字幕质量的能力。
- 如果功能依赖人声、字幕或剧集素材上下文，应提醒用户换成自己的视频，或先上传本地素材。
- 如果该 URL 后续不可访问，应让用户提供自己的视频，或重新选择一个可直接访问的 `.mp4` URL。

## Agent 使用规则

- 只有当用户明确要体验功能且没有素材时，才主动提到这个 URL。
- 生成请求示例时，把它标注为“演示视频 URL，可替换”。
- 不要将该 URL 写进固定脚本默认值。
- 调用前可用 `curl -I` 或等价方式确认 `Content-Type` 为 `video/mp4` 且 HTTP 状态成功。
